# PNU QuestMate V5 — 고난도 스탬프 인증형

AI 캠퍼스 퀘스트 + QR 2단계 인증 + 스탬프 리워드 프로토타입입니다.

## V5 핵심 강화점
- 입장 QR + 퇴장 QR 2단계 인증
- 날짜 기반 현장 코드
- 현장 확인 문구 입력
- 최소 체류시간: 발표 압축 25초 / 실제 운영 10분
- 하루 최대 2개 스탬프
- 동일 장소 하루 1회
- 인증 간격 45분
- 인증 로그/리워드 신청서 JSON 다운로드

## Netlify 배포
GitHub 저장소에 이 폴더 안의 파일을 올리고 Netlify에서 해당 저장소를 연결합니다.

중요 파일 위치:
```
netlify/functions/pnu-data.js
netlify/functions/checkin.js
```

태블릿 업로드 때문에 `pnu-data.js`, `checkin.js`가 루트에 올라가면 각각 위 경로로 파일명을 바꿔야 합니다.

## 발표 문장
기존 부산대 앱은 조회형 서비스입니다. QuestMate V5는 AI가 캠퍼스 미션을 생성하고, 실제 수행 여부를 입장 QR·퇴장 QR·현장코드·체류시간·하루 제한으로 검증해 스탬프와 학교 리워드/마일리지로 연결하는 참여형 플랫폼입니다.
